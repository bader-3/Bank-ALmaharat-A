# Arab Skills Bank (ASB)

## Project Overview

Arab Skills Bank is a modern educational platform built around a new concept called the **Learning Hours Economy**.

Instead of purchasing individual courses, learners purchase learning hours that can be used across multiple courses, instructors, and learning paths.

The platform aims to make learning more flexible, fair, and personalized.

This project is currently being developed as a high-quality prototype for a Hackathon, but every engineering decision should assume the platform will later evolve into a real production product.

---

# Vision

To become the leading Arabic learning platform that empowers learners through personalized education rather than traditional course marketplaces.

The platform focuses on skills, growth, and learning journeys instead of selling isolated courses.

---

# Core Innovation

The biggest innovation is the **Learning Hours Economy**.

Traditional platforms sell courses.

Arab Skills Bank sells learning time.

Learners purchase learning hours.

Those hours can then be invested in any compatible course available on the platform.

This creates flexibility and removes the psychological barrier of committing to expensive individual courses.

---

# AI Philosophy

Artificial Intelligence exists to help learners.

Its role is to:

- Understand learner goals.
- Build a learning profile.
- Recommend learning paths.
- Suggest skills to acquire next.
- Help learners stay motivated.

AI should never manipulate users or favor specific instructors.

Recommendations must remain objective and educational.

---

# Fairness Philosophy

Trainer fairness is one of the project's strongest principles.

The platform intentionally avoids:

- Best Seller labels
- Most Popular badges
- Top Rated rankings
- Student count comparisons

Learners should choose courses based on relevance to their goals—not popularity.

---

# Target Audience

The platform is designed for:

- University students
- Graduates
- Professionals
- Career changers
- Lifelong learners
- Trainers

The interface should be approachable for beginners while maintaining a premium experience.

---

# User Journey

The intended learning journey is:

1. Discover Arab Skills Bank.
2. Create an account.
3. Complete the AI Interview.
4. Build a personalized learning profile.
5. Receive AI-powered learning recommendations.
6. Purchase learning hours.
7. Start learning.
8. Track progress.
9. Complete courses.
10. Continue lifelong learning.

---

# Product Personality

The product should feel:

- Intelligent
- Calm
- Trustworthy
- Modern
- Premium
- Minimal
- Professional
- Human-centered

Never feel overwhelming.

Never feel cluttered.

---

# Visual Identity

The interface should prioritize:

- Clean layouts
- Spacious design
- Strong typography
- Premium spacing
- Consistent components
- Smooth animations

Avoid excessive visual effects.

The interface should resemble a modern SaaS product rather than a traditional education website.

---

# Language

Current prototype:

Arabic only.

The architecture should remain ready for English in the future.

When users click the English button, display a professional "Coming Soon" dialog.

Do not implement English pages during the prototype stage.

---

# Prototype Scope

The prototype focuses on demonstrating:

- Learning Hours Economy
- AI Interview
- Personalized Learning
- Learning Dashboard
- Course Discovery
- Learning Experience
- Trainer Fairness

Features intentionally postponed:

- Full English localization
- Payment gateway
- Email services
- Notifications
- Advanced analytics
- Production AI infrastructure

---

# Engineering Goal

Build the prototype using production-quality engineering practices.

The project should be:

- Clean
- Scalable
- Maintainable
- Modular
- High performance
- Accessible
- Ready to evolve into an MVP after the Hackathon.

Every technical decision should support long-term growth rather than short-term convenience.

---

# Success Criteria

A successful implementation should make first-time users immediately understand:

- What Arab Skills Bank is.
- Why Learning Hours are different.
- How AI helps personalize learning.
- Why trainer fairness matters.
- Why this platform is different from traditional course marketplaces.

If those ideas are communicated clearly, the project has achieved its primary objective.
